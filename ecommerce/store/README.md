# 🛒 Django E-Commerce Project

## 📌 About Project
This is a simple e-commerce website built using Django.  
Users can browse products, add to cart, and place orders.

---

## 🚀 Features

- User Registration & Login
- Product Listing
- Shopping Cart
- Checkout System
- Order Placement
- Order History
- Order Status Tracking
- Admin Panel

---

## 🛠 Tech Stack

- Django (Python)
- HTML, CSS, Bootstrap
- SQLite Database

---

## 🔄 Flow

Home → Product → Cart → Checkout → Order → Success → My Orders

---

## ⚙ How to Run

Open terminal and run:

```bash
pip install django
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
