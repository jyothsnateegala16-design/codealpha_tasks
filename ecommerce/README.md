# 🛒 CodeAlpha E-Commerce Website

## 📌 Project Overview

E-Commerce Website is a full-stack online shopping web application developed using **Django**.

The platform allows users to browse products, search items, add products to cart, and place orders through a simple and user-friendly interface.

This project was developed as part of the **CodeAlpha Full Stack Development Internship**.

---

## 🚀 Features

### 👤 User Authentication

* User Registration
* User Login and Logout
* Secure authentication system

### 🛍️ Product Management

* Display products with images
* Product categories
* Product details view
* Product search functionality
* Admin product management

### 🛒 Shopping Cart

* Add products to cart
* Update product quantity
* Remove products from cart
* View cart summary

### 📦 Order Management

* Checkout system
* Place orders
* Manage customer orders
* Order history

### 🎨 User Interface

* Responsive design
* Modern e-commerce style UI
* Attractive homepage banner
* Clean product cards
* Bootstrap-based layout

## ⚙️ Installation and Setup

### 1. Clone the repository

```bash
git clone https://github.com/jyothsnateegala16-design/codealpha-ecommerce.git
```

### 2. Navigate to project folder

```bash
cd codealpha-ecommerce
```

### 3. Create virtual environment

```bash
python -m venv venv
```

### 4. Activate virtual environment

Windows:

```bash
venv\Scripts\activate
```

### 5. Install dependencies

```bash
pip install -r requirements.txt
```

### 6. Apply migrations

```bash
python manage.py migrate
```

### 7. Run the server

```bash
python manage.py runserver
```

Open in browser:

```
http://127.0.0.1:8000/
```
